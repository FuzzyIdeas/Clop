## Features

- **Redesigned floating result**: every optimised file shows as a thumbnail card with a hover overlay of action buttons, audio files included (with album cover art when the file has it)
- **Secure send link expiration**: choose how long a "Send securely" link stays alive, from `1m` up to `3d`, or never
  > Pick it as you send (a slider on the floating result, a popover in compact mode) or change it later from the live link button.
- **Audio cover art**: choose whether embedded album art is optimised, removed, or left untouched, instead of always being dropped
  > Optimising recompresses the artwork at aggressive settings while keeping its resolution, so it stops bloating the file.
- **New Menubar icon**: there's also a picker if you want to switch back to the classic one
- **QuickLook** a hovered floating result by pressing `Space`
- **Inline pipeline scripts**: run shell code straight from a step with `runScript(code: "sips -Z 800 $1")`, no separate script file needed
- **`clop pipeline prompt`**: print a ready-to-paste reference of the whole pipeline language for an AI assistant, so you can describe a task and get a working pipeline back
  > Append your request as the last argument and add `-c` to copy the whole thing to the clipboard.

## Fixes

- `--copy` always copies a file path, even when optimisation or conversion fails (it copies the original), so you get back as many files as you put in
- The close (x) button reliably dismisses a floating result
- *Optimised file location* is respected when re-optimising from the floating result
- Audio bitrate is never raised above the file's original, even when the configured bitrate is higher
- Aggressive optimisation on audio steps the bitrate down a level instead of doing nothing
- Files worked on with an `Option`-drag copy keep their real name instead of a temporary `clop-dropzone-…` name

## Improvements

- Menubar menu hides keyboard shortcut hints for actions whose hotkeys are turned off
